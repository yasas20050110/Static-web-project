# 💻 Laptop Store Website

A responsive **Laptop Store** static website developed as an individual assignment for the **Internet and Web Technologies (ICT 142-3)** course at the **Faculty of Technological Studies, Uva Wellassa University of Sri Lanka**.

---

## 📖 Project Overview

The Laptop Store website is a modern and responsive static website developed using **HTML5**, **CSS3**, and **JavaScript**. It provides users with information about laptop products, services, company details, and contact information through a clean and user-friendly interface.

The project demonstrates the fundamental concepts of front-end web development, responsive web design, and organized project structure.

---

## 🎯 Objectives

- Develop a responsive static website.
- Apply HTML5, CSS3, and JavaScript concepts.
- Create a modern and user-friendly interface.
- Implement responsive web design.
- Organize project files using a structured folder layout.
